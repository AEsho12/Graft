# Contributing to Graft

Thanks for contributing.

## Ground Rules

- Be respectful and constructive.
- Keep pull requests focused and small.
- Do not include secrets, keys, tokens, or private infrastructure details.
- Do not submit reverse-engineered or proprietary third-party code.

## Development

```bash
npm install
npm run dev
```

## Pull Requests

Before opening a PR:

1. Run `npm run build`
2. Ensure no secrets are included
3. Add or update relevant docs
4. Describe the user-facing impact clearly

## Scope of This Public Repo

Accepted:

- UI/UX improvements
- Desktop app reliability fixes
- Docs and examples
- Developer workflow improvements

Not accepted here:

- Private infrastructure changes
- Supabase project internals
- Secret-handling implementation details
